Example of publishing a slideshow cinematic based on Phaser

Key requirements for this assignment:

3+ scenes: marble collision, splash impact, studio logo reveal, gameplay screen preview
2+ audio assets: splash collision sound, soda bubble sound
2+ image assets: marble sprites, splash images
2+ geometric shapes: circles and rectangular UI elements
1+ logotype: Marble Searealm
1+ multi-line text: gameplay screen includes multi-line text
1+ fading animation: studio name fades in/out in the second scene
1+ object motion animation: marbles move along a semicircular path and collide at the center
